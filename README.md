Projet Application Android - Gestion des Scores

Ce projet est une application Android développée en C# (MAUI.Net) qui permet de gérer les scores des equipes dans différents sports (ici pour les tests, du Quidditch). 
L'application offre une interface utilisateur simple pour ajouter, afficher et supprimer des equipes ou des matchs.
Fonctionnalités principales :

1. Ajouter une équipe avec son nom et son score initial.
2. Afficher la liste des équipes avec leurs scores.
3. Supprimer une équipe de la liste.
4. Ajouter un match entre deux équipes et mettre à jour leurs scores en fonction du résultat.
5. Afficher la liste des matchs joués avec les scores correspondants.

Technologies utilisées :
- C# avec MAUI.Net pour le développement de l'application Android.
- XAML pour la conception de l'interface utilisateur.
- SQLite pour la gestion des données locales (si applicable).

Instructions pour exécuter l'application :
1. Cloner le dépôt du projet depuis GitHub.
2. Ouvrir le projet dans Visual Studio avec le support MAUI installé.
3. Configurer un émulateur Android ou connecter un appareil Android pour le déploiement.
4. Construire et exécuter l'application.
5. Utiliser l'interface pour ajouter des équipes, gérer les scores et visualiser les matchs.

Remarques :
- Assurez-vous d'avoir les dernières versions de Visual Studio et du SDK MAUI installées.
- Le code source est bien commenté pour faciliter la compréhension et la maintenance.
